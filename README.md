# HUSTCSE software security

a simple front-end for security tools using layui

## file structure

```bash

controlCenter/
  └─
layui/
  ├─css
  │  └─layui.css   # 核心样式库
  ├─font
  ├─img
  │  └─avator.png
  └─layui.js       # 核心模块库
lib/
  └─
index.html         # 首页主体
index.js

```

## usage

just git clone and click index.html
